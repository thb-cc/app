
# Application

## Purpose

`thb-cc/app` is a simple joke application with the sole purpose of displaying random computer science memes on-demand.

## Technical Details

The application is written in the Java programming language, using the [Spring Boot](https://spring.io/projects/spring-boot) framework.
Its dependencies and lifecycle are managed with the build tool [Maven](https://maven.apache.org/).
You can find out which dependencies are in use by taking a look at the [pom.xml](https://github.com/thb-cc/app/blob/main/pom.xml) file.
